## Welcome to Scalabe Chat Engine!

To install:

```
yarn add @scalablechat/scalable-chat-engine
```

To update:
```
yarn remove @scalablechat/scalable-chat-engine

yarn add @scalablechat/scalable-chat-engine
```

To install with specific version
```
`yarn add @scalablechat/scalable-chat-engine@X.X.X
```
